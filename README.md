# PVZManager (WPF-приложение)
[![GitHub top language](https://img.shields.io/badge/language-C%23-blue)](LINK-НА-ВАШ-РЕПОЗИТОРИЙ)

## 📌 О проекте
**PVZManager** — это десктопное приложение для автоматизации работы пункта выдачи заказов. Система разработана в рамках курсового проекта.

### Основной функционал:
*   ✅ Приёмка и учёт заказов от курьерских служб.
*   ✅ Поиск и выдача заказов клиентам.
*   ✅ Оформление возвратов.
*   ✅ Учёт свободных ячеек хранения и инвентаризация.
*   ✅ Ролевая модель (администратор, оператор, кладовщик).
*   ✅ Формирование отчётов по продажам и возвратам.

## 🛠 Технологический стек
*   **Язык и платформа:** C#, .NET 8.0.
*   **Интерфейс:** Windows Presentation Foundation (WPF).
*   **База данных:** MS SQL Server.
*   **Доступ к данным:** ADO.NET (System.Data.SqlClient).
*   **Система контроля версий:** Git, GitHub.
